Erstellung von Filtern

Das Filter Dialogfenster läßt sich öffnen über die Menüleiste: Einstellungen > Filter oder die Tastenkombination Shift + F.

Filterübersicht:

  • Checkbox
    		• =Aktivierung (alle aktivierten Filter werden auf eingehende eMails geprüft)
  • Neu
    		• =neuen Filter anlegen (1. Filterdefinition)
  • Bearbeiten
    		• =bestehende Filter ändern (1. Filterdefinition)
  • Löschen
    		• =markierten Filter löschen
  • OK
    		• =alle Änderungen werden gespeichert
  • Abbrechen
    		• =alle Änderungen werden verworfen

Anmerkungen:

  • Filternamen müssen eindeutig sein.
  • Jeder Benutzer kann beliebig viele Filter für neue Konten definieren.

Filterdefinition:

Angaben
  • Filtername: Name des Filters
  • FilterAktion:
    1. Verschieben:
      		2. gefilterte eMails werden in Ordner "Filterergebnisse" gespeichert.
    2. Löschen:
      		3. gefilterte eMails werden *sofort* gelöscht
  • Verknüpfung:
    1. alle Bedingungen (AND):
      Filter greift nur, wenn alle Bedingungen erfüllt sind.
    2. mindestens 1 Bedingung (OR):
      Filter greift nur, wenn mindestens 1 Bedingung erfüllt ist.
    3. Bedingungen:
      4. Auswahl der Art:Betreff, Absender, Empfänder, Nachrichtentext
      Operator:enthält
      Wert:Eingabe durch Benutzer
      Sinnvoll ist eine Begrenzung auf max. 3 verschiedene Bedingungen.
